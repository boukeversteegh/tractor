---
title: XPath 2.0 Query Execution
priority: 1
---

Execute XPath 2.0 expressions against generated XML to find and extract nodes, attributes, or values.
Returns location information for matched elements.

XPath 2.0 functions available:
- String: ends-with(), starts-with(), matches() (regex), lower-case(), upper-case(), string-join()
- Sequences: for expressions, quantified expressions
- All standard XPath 2.0 operators and functions
