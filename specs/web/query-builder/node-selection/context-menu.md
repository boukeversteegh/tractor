title: Node Context Menu
priority: 2
---

Right-click context menu for node pill actions.

## Menu Options

- **Set as target / Unset as target**: Toggle explicit target status
- **Select / Deselect**: Toggle node selection

## Behavior

- Opens on right-click of node pill
- Closes on blur or after action
- Toggles with repeated right-clicks
- Actions update selection state immediately
