Terminal Compatibility

For terminal output, we need ANSI escape codes. Two options:

Option A: Basic 16 ANSI Colors

Works everywhere, but colors depend on user's terminal theme

0 Black
1 Red
2 Green
3 Yellow
4 Blue
5 Magenta
6 Cyan
7 White
8 Bright Black
9 Bright Red
10 Bright Green
11 Bright Yellow
12 Bright Blue
13 Bright Magenta
14 Bright Cyan
15 Bright White

Current mapping: Blue(kw), Cyan(type/fn), Yellow(str), Green(cmt), White(id), Dim(op)

Option B: True Color (24-bit RGB)

Exact colors, but requires modern terminal (most support it now)

\x1b[38;2;R;G;Bm  (e.g., \x1b[38;2;251;73;52m for Gruvbox red)

Supported: iTerm2, Windows Terminal, GNOME Terminal, Alacritty, Kitty, VS Code terminal
